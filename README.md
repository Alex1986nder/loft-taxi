Учебный проект - "Приложение для заказа такси"
•	Реализован следующий функционал: авторизация, заполнение "платежных" данных с валидацией, запрос на построение маршрута по заданным адресам, построение маршрута на карте. Адреса и данные для авторизации заданы на сервере. Флаг авторизации и "платежные" данные сохраняются в localstorage.

Стек технологий:
•	React.js
•	react-router
•	Redux
•	redux-saga
•	Material UI
•	redux-actions
•	CSS-modules
•	Mapbox API
•	Webpack
